# Desafio-Open-Banking-Compass
API Spring Boot de Produtos.
No projeto foi demonstrada uma arquitetura simples de microserviços responsável por manter os dados de um produto, ou seja, um CRUD.

* JAVA 11
* BANCO DE DADOS H2
* JPA
* SPRING BOOT
